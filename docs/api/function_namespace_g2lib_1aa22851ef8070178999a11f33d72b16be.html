void G2lib::FresnelCS(real_type y, real_type &C, real_type &S)¶

Purpose:

Compute Fresnel integrals C(x) and S(x)

\[ S(x) = \int_0^x \sin t^2 \,\mathrm{d} t, \qquad C(x) = \int_0^x \cos t^2 \,\mathrm{d} t \]
